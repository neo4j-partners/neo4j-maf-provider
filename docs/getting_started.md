# Getting Started

This guide covers setting up and using the Neo4j Context Provider with Microsoft Agent Framework.

## Installation

```bash
pip install agent-framework-neo4j
```

With Azure AI embeddings:

```bash
pip install agent-framework-neo4j[azure]
```
